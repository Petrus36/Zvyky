# Autoškola Website

Moderní webová stránka pro autoškolu postavená s TypeScript, React a Tailwind CSS.

## Technologie

- **React 18** - UI knihovna
- **TypeScript** - Typovaný JavaScript
- **Vite** - Rychlý build tool
- **Tailwind CSS** - Utility-first CSS framework
- **React Router** - Routování

## Instalace

1. Nainstalujte závislosti:
```bash
npm install
```

2. Spusťte vývojový server:
```bash
npm run dev
```

3. Otevřete prohlížeč na `http://localhost:5173`

## Build

Pro produkční build:
```bash
npm run build
```

## Struktura projektu

```
src/
  components/     # Znovupoužitelné komponenty
    Navigation.tsx
    Footer.tsx
  pages/         # Stránky aplikace
    Home.tsx
  App.tsx        # Hlavní komponenta
  main.tsx       # Vstupní bod
  index.css      # Globální styly s Tailwind
```
